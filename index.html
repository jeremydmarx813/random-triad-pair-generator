Triad Pair Generator

Welcome to the Random Triad Pair Generator!
This is a practice tool designed to randomize the different combinations of ways you can apply hexatonic scales or triad pairs.

  • Set the vibe with the given key and mode, perhaps using a loop pedal
  • Find the two triads that fit in that mode/shade given the triad pair constraints
  • Practice that triad pair using the given melodic and rhthmic patters. The rythym can be a subdivision or grouping - its your pick!
  • Click the lock checkbox to keep that variable constant!
